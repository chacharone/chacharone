## top

<br>

+ top 명령어는 현재 os의 상태를 나타내주는 명령어
+ 메모리 사용량, CPU 사용량 등을 나타낸다.
+ top를 실행하는 동안에 주기적인 업데이트로 실시간에 근접한 내용을 보여준다.

<br>

+ 제일 왼쪽 위에 있는 것이 시간이다. 그 오른쪽에 있는 것은 os가 살아 있는 시간이고 그 다음이 현재 접속중인 유저 세션 수 이다.
